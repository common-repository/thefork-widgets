Embed the official TheFork widget via shortcode.

== Description ==

This plugin enables you to easily embed the official TheFork reservation widget (https://www.theforkmanager.com/widgets/) into your WordPress site via shortcode. It provides you an easy configuration page in order to generate the shortcode.

== Installation ==
1. Upload and activate the plugin.

== Frequently Asked Questions ==
<strong>How do I config the widget?</strong>
The widget selected is already preconfigurated. You just should paste the shortcode in the edition page.

New feature. You can also add a floating button. Shortcodes are not necessary. 


<strong>Where do I get my restaurant ID?</strong>
You can visit the tab \"Get your ID\". This part shows you how to get the ID.

<strong>How many shortcodes can I generate?</strong>
All of you want.
